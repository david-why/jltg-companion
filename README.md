# JLTG Companion App

A companion web app that lets you play the Hide and Seek game with your friends.

_Note: This app is not endorsed by JLTG in any way, shape, or form. All information is sourced from the internet._

## Setup instructions

To start a development server, run:

```sh
bun i
bun dev
```

To build for production and deploy to Cloudflare:

```sh
bun i
bun run build
bunx wrangler deploy
```
